## 汽水音乐

嘿嘿 假的

这个只是个放汽水音乐布局做的一个音乐 app

未 设置登录接口 如果请求不到 可以先去个人中心 点击游客

但是那玩意 忘记上传 github 了 现在用 vue3 写一下

一个是熟悉一下 vue3 组合 api

另一个是 看看 vuex 和 pinia 使用上有啥区别啥的
